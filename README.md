# Caya
Caya is the best Instant Messaging client for Haiku.

[Project Page](http://www.versut.com/caya)

Protocols supported:

* XMPP (Jabber, GTalk)
* Yahoo
* AIM

How to install
--------------

The latest stable version is available as package through HaikuDepot.

How to build
------------

./configure

jam
